# Relational Database Schema In Music Industry Context  
## A database and its schema designed from scratch for fast SQL querying using MySQL.  

### Synopsis  
A database schema was designed consisting of 12 tables to be used in music industry setting. Data was normalized for efficient SQL queries. An entity relationship diagram defined all relationships between the entities. phpMyAdmin platform using MySQL database was used to set up the actual database ready for CRUD operations using SQL.
